WaveDiff is a differentiable data-driven wavefront-based PSF modelling framework developed in the CosmoStat lab at CEA Paris-Saclay.

This documentation provides information for the installation and running of WaveDiff. If you are unable to find what you are looking for here, please open an issue on the GitHub repository.

If you use WaveDiff for your academic work, we ask you to please cite Liaudat et al. [2023].
