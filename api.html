API Reference

This section contains the API reference for the main packages in WaveDiff.

wf_psf.data

Data handling and preprocessing utilities.

wf_psf.metrics

Evaluation metrics and performance measurement utilities.

wf_psf.plotting

Visualization tools and plotting utilities for PSF data.

wf_psf.psf_models

Implementation of PSF model classes and TensorFlow utilities.

wf_psf.run

Entry point and application runner.

wf_psf.sims

Simulation tools and data generation utilities.

wf_psf.training

Training routines and utilities for PSF models.

wf_psf.utils

Utility functions and helpers.
